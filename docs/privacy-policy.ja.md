# プライバシーポリシー

**最終更新日: 2025-06-19**

## 概要

ccpocket（以下「本アプリ」）は、セルフホストのBridge Serverに接続してコーディングエージェント（Claude Code、Codex）を操作するモバイルクライアントです。本アプリはプライバシーを重視して設計されており、データはお使いのデバイスとサーバーに保持されます。

## データ収集

### 収集するデータ

- **Firebase匿名認証**: プッシュ通知を有効にするためにFirebase匿名認証を使用します。匿名ユーザーIDが生成されますが、個人情報（メールアドレス、氏名、電話番号）は収集しません。
- **FCMトークン**: プッシュ通知用のデバイストークンがFirebase Firestoreに保存されます。匿名ユーザーIDとのみ関連付けられます。

### 収集しないデータ

- 個人情報（氏名、メールアドレス、電話番号等）
- 会話内容やセッションデータ
- 利用分析やトラッキング
- 位置情報
- 連絡先、写真、その他のデバイスデータ

## データの保存

- **お使いのデバイス上**: 接続設定、セッション履歴、プロンプト履歴はSharedPreferencesおよびSQLiteを使用してデバイスにローカル保存されます。
- **お使いのBridge Server上**: セッションデータと会話履歴はセルフホストのBridge Serverに保存されます。このサーバーはお客様が完全に管理します。
- **Firebase**: 匿名ユーザーIDとFCMトークンのみがプッシュ通知配信のためにFirebase Firestoreに保存されます。

## プッシュ通知

Bridge Serverに更新がある場合（ツール承認リクエスト、セッション結果等）、Firebase Cloud Messaging (FCM) を通じてプッシュ通知が送信されます。通知内容はFirebase Cloud Functions経由で中継されます。プッシュ通知はアプリ設定でサーバーごとに無効化できます。

## サードパーティサービス

| サービス | 目的 | 共有データ |
|---------|------|-----------|
| Firebase Authentication | プッシュ通知用匿名認証 | 匿名ユーザーID |
| Firebase Cloud Messaging | プッシュ通知配信 | FCMデバイストークン |
| Firebase Firestore | FCMトークン保存 | 匿名ユーザーID + FCMトークン |

広告主やデータブローカーとのデータ共有は行いません。

## データの保持

- **ローカルデータ**: アプリをアンインストールするか、アプリデータを消去するまで保持されます。
- **Firebaseデータ**: 匿名認証とFCMトークンは、トークンの有効期限切れまたはアプリのアンインストール時に自動的にクリーンアップされます。

## お客様の権利

- アプリ設定またはデバイス設定でいつでもプッシュ通知を無効化できます。
- アプリをアンインストールすることですべてのローカルデータを削除できます。
- Bridge Serverはお客様が管理しており、サーバー側のデータはいつでも削除できます。

## お子様のプライバシー

本アプリは13歳未満のお子様を対象としておらず、お子様からのデータを意図的に収集することはありません。

## ポリシーの変更

本プライバシーポリシーは随時更新される場合があります。変更はこのページに更新日とともに掲載されます。
