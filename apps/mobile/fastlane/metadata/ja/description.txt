ccpocketは、Claude CodeとCodex — コードを書き、編集し、管理するAIコーディングエージェント — のモバイルリモコンです。

本アプリはAnthropic・OpenAIとは無関係であり、両社の公認・推薦を受けたものではありません。

どこからでもエージェントを操作
ファイル編集、コマンド実行、ツール呼び出しの承認をスマホから。デスクに縛られず、移動中でもコーディングセッションを管理できます。

リアルタイムストリーミング
エージェントの思考とコーディングをリアルタイムで確認。インラインdiff、思考ブロック、構造化プランをライブで表示します。

マルチセッション管理
複数セッションの実行と切り替え。過去の会話の再開、履歴検索、プロジェクト別フィルタリングに対応。

高機能Diffビューア
シンタックスハイライト付きのdiffでコード変更をレビュー。Hunk選択、ファイル統計、エージェントへのフィードバック送信が可能です。

柔軟な接続方法
保存済みマシン、QRコード、mDNS自動検出、手動URLの4つの方法で接続。SSH経由でのリモートサーバー管理にも対応。

プッシュ通知
エージェントが承認を必要とする時やタスク完了時に即座に通知。承認リクエストを見逃しません。

プライバシー最優先
コードと会話はあなたのサーバーに保管。テレメトリなし、トラッキングなし。個人データの収集は一切行いません。

利用要件
- 開発マシン上で稼働するccpocket Bridge Server
- サーバーにインストール済みのClaude CodeまたはCodex CLI
- スマホとサーバー間のネットワーク接続（ローカルネットワークまたはTailscale推奨）
